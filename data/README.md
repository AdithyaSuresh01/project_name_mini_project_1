# Data Directory

## iris.csv

The Iris dataset containing measurements for 150 iris flowers across 3 species:

**Columns:**
- sepal_length (cm)
- sepal_width (cm) 
- petal_length (cm)
- petal_width (cm)
- species (setosa, versicolor, virginica)

**Source:** Fisher, R.A. (1936). The use of multiple measurements in taxonomic problems.
