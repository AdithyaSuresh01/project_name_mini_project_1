# IrisStatsNotebook

A data science project analyzing the Iris dataset with modular Python code and Jupyter notebooks.

## Project Structure

- **notebooks/**: Jupyter notebooks for exploratory analysis
- **src/**: Reusable Python modules for data loading, statistics, and plotting
- **data/**: Dataset files
- **reports/**: Generated figures and documentation

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the notebook:
   ```bash
   jupyter notebook notebooks/mini_project_1_iris_stats.ipynb
   ```

## Usage

The main analysis is in `notebooks/mini_project_1_iris_stats.ipynb`, which uses the modular code from the `src/` directory.
